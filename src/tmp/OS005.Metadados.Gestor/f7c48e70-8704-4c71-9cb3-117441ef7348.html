Declaração 

Ao executar os arquivos adquiridos via internet deverá criada uma estrutura de diretórios na unidade “C” do tema para download. Essa estrutura de diretórios permitirá o usuário abrir o projeto ArcGis para download. Caso o usuário queira utilizar outra unidade, poderá fazê-lo, entretanto, ao executar o projeto, deverá apontar onde está cada arquivo solicitado, ou editar os caminhos dos arquivos. É importante que seja mantida a estrutura de projeto, pois o mesmo possui uma apresentação preliminar das informações básicas, tais como cores.

Deverá ser criada uma pasta, na raiz do disco “C”, denominada “AmazoniaLegal”. Lá serão armazenados resultados da descompactação, da seguinte forma:
